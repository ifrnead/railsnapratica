Criando um projeto Rails localmente

Para criar um novo projeto com Ruby on Rails é bem simples, basta utilizar o comando rails new <nome-do-projeto>, substituindo <nome-do-projeto> pelo nome do projeto que você deseja criar. Vamos criar um novo projeto utilizando o comando abaixo.

  
    rails new meuprojeto

Ao executar o comando acima, uma série de mensagens serão exibidas na tela, conforme ilustrado abaixo.

  
    create
    create README.rdoc
    create Rakefile
    create config.ru
    create .gitignore
    create Gemfile
    create app
    create app/assets/javascripts/application.js
    app/assets/stylesheets/application.css
    app/controllers/application_controller.rb
    [...]
    run bundle install
    Fetching gem metadata from https://rubygems.org/...........
    Fetching version metadata from https://rubygems.org/...
    Fetching dependency metadata from https://rubygems.org/..
    Resolving dependencies............................................
    Installing rake 11.1.2
    Using i18n 0.7.0
    Installing json 1.8.3 with native extensions
    [...]
    Bundle complete! 12 Gemfile dependencies, 55 gems now installed.
    Use `bundle show [gemname]` to see where a bundled gem is installed.

As primeiras mensagens que aparecem, precedidas de create, indicam que um diretório ou um arquivo está sendo criado pelo Rails para compor a estrutura de diretórios e artefatos do seu novo projeto. Além disso, um comando é executado automaticamente: bundle install. Conforme explicado anteriormente, esse comando lê o Gemfile, baixa e instala as Gems necessárias para o projeto. Quando a execução do comando for concluída, significa que o seu novo projeto já está criado.

Executando seu projeto localmente

Para executar o seu projeto e poder visualizá-lo no navegador, primeiramente vá para o terminal e digite o comando: rails server (ou apenas rails s), conforme ilustrado abaixo.

  
    $ rails server
    => Booting WEBrick
    => Rails 4.2.4 application starting in development on http://localhost:3000
    => Run `rails server -h` for more startup options
    => Ctrl-C to shutdown server
    [2016-05-14 17:38:58] INFO WEBrick 1.3.1
    [2016-05-14 17:38:58] INFO ruby 2.2.3 (2015-08-18) [i686-linux]
    [2016-05-14 17:38:58] INFO WEBrick::HTTPServer#start: pid=4180 port=3000

Ao executar esse comando, um servidor web de desenvolvimento chamado WEBrick será inicializado na porta 3000, portanto já podemos acessar o endereço http://localhost:3000 no navegador para ver o nosso projeto em funcionamento. Portanto, abra o navegador e acesse o referido endereço. Ao fazer isso, a página ilustrada na Figura 1 deverá ser exibida.

Parabéns! Você acabou de criar o seu primeiro projeto em Ruby on Rails! É claro que ele ainda está vazio, mas esse é o primeiro passo para criar aplicações mais complexas.
