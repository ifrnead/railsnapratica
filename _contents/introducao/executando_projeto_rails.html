Executando seu projeto no Cloud9

Executando seu projeto localmente

Para executar o seu projeto e poder visualizá-lo no navegador, primeiramente vá para o terminal e digite o comando: rails server (ou apenas rails s), conforme ilustrado abaixo.

  
    $ rails server
    => Booting WEBrick
    => Rails 4.2.4 application starting in development on http://localhost:3000
    => Run `rails server -h` for more startup options
    => Ctrl-C to shutdown server
    [2016-05-14 17:38:58] INFO WEBrick 1.3.1
    [2016-05-14 17:38:58] INFO ruby 2.2.3 (2015-08-18) [i686-linux]
    [2016-05-14 17:38:58] INFO WEBrick::HTTPServer#start: pid=4180 port=3000

Ao executar esse comando, um servidor web de desenvolvimento chamado WEBrick será inicializado na porta 3000, portanto já podemos acessar o endereço http://localhost:3000 no navegador para ver o nosso projeto em funcionamento. Portanto, abra o navegador e acesse o referido endereço. Ao fazer isso, a página ilustrada na Figura 1 deverá ser exibida.

Parabéns! Você acabou de criar o seu primeiro projeto em Ruby on Rails! É claro que ele ainda está vazio, mas esse é o primeiro passo para criar aplicações mais complexas.
